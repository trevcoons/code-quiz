# code-quiz

This site is a code quiz where the user will be tested on their knowledge about code. The quiz is timed and deducts time for incorrect answers. Users can save their score in the highscores list.

## Built With

* [HTML](https://developer.mozilla.org/en-US/docs/Web/HTML)
* [CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)
* [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
